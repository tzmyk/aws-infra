# コスト見積もり

> **見積もり基準:** 2026年4月時点の東京リージョン料金

---

## 月額コスト概算

| サービス | 構成 | 月額（USD） |
|---|---|---|
| EC2 App Server | t3.small × 2台 × 720h | $30 |
| EC2 管理EC2 | t3.micro × 1台 × 720h | $8 |
| RDS MySQL | db.t3.small Multi-AZ × 720h | $50 |
| ALB | $0.008/h × 720h + LCU | $10 |
| NAT Gateway | $0.045/h × 720h + データ転送 | $35 |
| S3 | ログ保存 5GB 程度 | $1 |
| CloudWatch Logs | ログ取り込み・保存 | $3 |
| ACM | SSL 証明書 | **無料** |
| **合計** | | **≈ $137** |

> **NAT Gateway が高い:** $35/月と全体の 25%。  
> S3 へのアクセスはゲートウェイ型 VPC エンドポイント（無料）を使うと数ドル削減できる。

---

## 研修環境のコスト削減

本番相当の構成を研修で常時起動していると費用がかさむ。以下の削減策を取る。

| リソース | 本番想定 | 研修環境 | 削減額 |
|---|---|---|---|
| EC2 App Server | t3.small × 2 | t3.micro × 1 | -$22 |
| RDS MySQL | db.t3.small Multi-AZ | db.t3.micro シングルAZ | -$35 |
| NAT Gateway | 常時起動 | 常時起動（削れるが管理コスト増） | $0 |
| **研修環境合計** | **$137** | **≈ $50** | |

> **研修終了後はリソースを削除すること。**
> `terraform destroy` で一括削除できる。削除し忘れると課金が続く。

---

## コスト異常検知

```
設定するアラート（AWS Budgets）:
  予算: $60/月（研修環境）
  通知: 80% 到達（$48）で メール通知
```

---

## 設計上の禁止事項

- 研修終了後に `terraform destroy` を忘れない
- RDS のインスタンスクラスを db.r5 など大きいサイズにしない
- CloudWatch Logs の保持期間を「失効なし」に設定しない（必ず期間を設定する）
